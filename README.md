# laba4_tg_bot
Мой телеграм-бот представляет из себя определённый фильтр, который применяется к сайту Market Dota2. Он способен найти самые дешёвые лоты всех тринадцати предметов, имеющих редкость Arcana, которые выставлены на продажу на этом сайте. код бота состоит из четырёх файлов:
В файле dota2market_bot.py находиться скелет телеграм-бота.
